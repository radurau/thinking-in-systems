# Thinking in Systems

An illustrated, self-contained HTML presentation — *A Primer on Seeing the Invisible* — with companion speaker notes.

## Contents

| File | Description |
| --- | --- |
| [`index.html`](./index.html) | **Landing page (entry point).** Lets you choose what to open in this browser: the presentation or the control panel. |
| [`presentation.html`](./presentation.html) | The full-screen slide deck. Single self-contained HTML file (inline CSS/JS, no build step). |
| [`thinking-in-systems-notes.html`](./thinking-in-systems-notes.html) | The control panel — presenter view: speaker notes per slide, live section highlight, and a timer. Links jump the deck. |

## Viewing

All files are standalone — just open them in a browser:

```sh
open index.html
```

`index.html` is a chooser; from there pick **Presentation** (`presentation.html`) or
**Control Panel** (`thinking-in-systems-notes.html`). You can also open either directly.

No dependencies, no build, no server required.

> **Note:** the deck uses animation. If you have macOS *Reduce Motion* enabled, some
> transitions are intentionally simplified to avoid a strobe effect.
